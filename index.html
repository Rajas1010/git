# Command Description Usage
1 git --version "To check whether Git is installed or not." Shows the installed Git version, e.g., git version 2.38.1.windows.1 (in cmd)
2 mkdir github && cd github "Open the command prompt and create a directory using the 'mkdir' command (e.g., mkdir github). Then navigate to that directory using the 'cd' command (e.g., cd github)."
3 git init "git init" is used to turn a directory into a Git repository. Generates a '.git' folder within the specified directory.
4 git status "To view the status of the working directory and staged changes in the repository." (in cmd) On branch main (or) branch name       No commits yet
      Nothing to commit (or) Changes to be committed:
5 git config --global user.name "To view the globally configured Git username." Displays the globally configured Git username.
6 git config --global user.email "To view the globally configured Git email." Displays the globally configured Git email.
7 git config --global --unset user.name "To remove the globally configured Git username." Removes the globally configured Git username.
8 git config --global --unset user.email "To remove the globally configured Git email." Removes the globally configured Git email.
9 git add . "To stage changes for commit in Git." Adds the changes in your working directory to the staging area for the next commit.
10 git commit -m "FisrtCommit" "To record changes made to the repository with a message describing the changes." (in cmd) [master (root-commit) c8595da] FisrtCommit
1 file changed, 1 insertion(+)
create mode 100644 hello.py

12 git config --global user.email "Rajasekar.Rj100@gmail.com" "To set the globally configured Git email." Use the 'git config --global user.email' command it shows your configured user email
13 git log "To display the commit history of the repository." Shows the commit history of the repository.
Git Branch Commands
14 git branch "To list all branches in the repository." Displays a list of all branches in the repository.
15 git branch NewBranch "To create a new branch named 'NewBranch'." Creates a new branch called 'NewBranch'.
16 git checkout NewBranch "To switch to the 'NewBranch' branch." Switched to branch 'NewBranch'
Changes the active branch to 'NewBranch'.
17 git checkout main "To switch back to the 'main' branch." Changes the active branch back to 'main'.
18 git merge NewBranch "To merge changes from 'NewBranch' into the current branch." Merges the changes from 'NewBranch' into the current branch.
19 git branch -d NewBranch "To delete the 'NewBranch' branch (if merged)." Deletes the 'NewBranch' branch if it's already merged.
20 git checkout -b SecondBranch "To create and switch to a new branch named 'SecondBranch'." Creates and switches to a new branch called 'SecondBranch'.
21 git branch -D SecondBranch "To forcefully delete the 'NewBranch' branch (regardless of merge status)." Deletes the 'NewBranch' branch forcefully.
Advanced Commands
22 echo.>f1 "To Create empty file" File created in the corresponding directory
23 dir /b "View the list of files" List of files
24 type f1.txt "To display the contents of the 'f1.txt' file" Displays the contents of the 'f1.txt' file.
25 git cherry-pick "To temporarily save changes in a stash, including untracked files." Stashes changes, including untracked files, for later retrieval.
26 git stash "To temporarily save changes in a stash, including untracked files." Stashes changes, including untracked files, for later retrieval.
27 git stash list "To list all stashes created in the repository." Displays a list of all stashes in the repository.
28 git stash show "To show the changes stored in the most recent stash." Displays the changes stored in the most recent stash.
29 git stash apply "To apply the changes from the most recent stash." Applies the changes from the most recent stash to the working directory.
30 git restore --staged f2 "To unstage changes from the staging area." Removes changes from the staging area, but preserves them in the working directory.
31 git stash pop 0 "To apply and remove the most recent stash." Applies and removes the most recent stash from the list of stashes.
32 git log --oneline "To display a compact version of the commit history." Displays a concise commit history with each commit on a single line.
33 git revert 7af537f "To create a new commit that undoes the changes introduced by the specified commit." Creates a new commit that reverts the changes made by commit '7af537f'.
34 git revert HEAD "To create a new commit that undoes the changes introduced by the last commit." Creates a new commit that reverts the changes made by the last commit.
Remote Commands
35 git remote add origin https://github.com/Rajas1010/Commands.git "To associate the remote repository URL with the name 'origin'." Associates the remote repository URL with the name 'origin'.
36 git remote -v "To view remote repositories linked to the current repository." Displays a list of remote repositories associated with the current repository.
37 git clone https://github.com/Rajas1010/Commands.git "To clone a remote repository to your local machine." Creates a local copy of the remote repository.
38 git pull -r "To fetch and rebase changes from the remote repository." Fetches changes from the remote repository and rebases the current branch.
39 git pull origin main "To fetch and merge changes from the remote 'main' branch." Fetches and merges changes from the remote 'main' branch into your local branch.
40 git push origin main "To push committed changes to a remote repository (e.g., on GitHub) in the 'main' branch." Uploads the local commits to the remote repository's 'main' branch.
